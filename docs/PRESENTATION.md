# üìù APPENDIX: Common Questions & Answers

**Q: What is the Silhouette Score?**
> A: It measures how well customers fit their assigned cluster vs other clusters. Range is -1 to +1. Above 0.5 is good, above 0.7 is excellent.

**Q: Why Hierarchical Clustering instead of K-Means?**
> A: Hierarchical gives us the dendrogram visualization, showing cluster relationships. It's also deterministic - same data always gives same results.

**Q: What data format is required?**
> A: CSV files with headers. Numeric and categorical columns are auto-detected.

**Q: Can it handle missing values?**
> A: Currently, missing values should be handled before upload. Future versions will include imputation.

**Q: How many customers can it handle?**
> A: Tested up to 10,000 customers. For larger datasets, use the PCA option to reduce dimensions.

**Q: Is customer data secure?**
> A: Data stays on your server. Nothing is sent to external services. For production, add authentication.
